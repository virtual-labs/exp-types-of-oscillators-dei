<h3 style="color:#2867B2;">3. To Study Forced Oscillatory Motion</h3>
<ul>
  <li><p><strong>Step 1:</strong> Reset the simulation again.</p></li>
  <li><p><strong>Step 2:</strong> Set the damping coefficient (b) &gt; 2&radic;2.</p></li>
  <li><p><strong>Step 3:</strong> Change the value of initial velocity v<sub>0</sub> = 2 m/sec.</p></li>
  <li><p><strong>Step 4:</strong> Set value of driven angular frequency (&omega;) = 1.41.</p></li>
  <li><p><strong>Step 5:</strong> Set some values of driven amplitude F<sub>0</sub> which will be the force for oscillation.</p></li>
  <li><p><strong>Step 6:</strong> Drag the ball to the maximum starting position, say x<sub>0</sub> = 10.0 m, and play the animation button.</p></li>
  <li><p><strong>Step 7:</strong> Observe the graph until the animation ends.</p></li>
</ul>
